Entrar a CMD o en la Terminal

1.- Correr 'npm install' instalar paquetes
2.- Correr 'npm start' iniciar el servidor con puesto 3000
3.- Correr 'npm run build' general la compilación de código


Carpetas y archivos principales:
- archivo Header.js: Se definen los items del menu
- archivo route.js: se incluyen las vistas de la carpeta components, si no se hace no se mostrara la vista
- carpeta components: vistas, hacen uso de la carpeta service
- carpeta services: funciones para conectar a la api
